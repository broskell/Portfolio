# Dashboard App

This workspace will become the private admin dashboard for managing portfolio content.

It intentionally does not expose `dev` or `build` scripts in Milestone 1 because the dashboard UI is not implemented yet. Adding scripts before the app exists would create fake automation and make the workspace less trustworthy. The dashboard will receive its Vite, React Router, React Query, React Hook Form, Zod, DnD Kit, TipTap, and Tailwind setup in the dashboard milestones.
